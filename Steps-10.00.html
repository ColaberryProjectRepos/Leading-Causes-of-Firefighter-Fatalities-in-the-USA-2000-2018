Visual 1 : Shape Map

Download the doc on the top right corner to download the step by step by step process for visuals


To create visual 1 : follow the steps outlined below

-          Choose the visual Shape map

Fields Tab:

-          Location: State

-          Color Saturation: Fatalities/Year

Format Tab

-          General: X Position = 158, Y Position = 97, Width = 519, Height = 287

-          Data Colors: Diverging = on, Minimum = 7DD0E4, Center = 262A76, Maximum = 8E1F20; minimum = 0, Center = 2, Maximum = 3

-          Shape: Map USA: States

-          Default color: Show = on, color = C8C6C4, Border color = 605E5C, Border Thickness = 1

-          Zoom = Off

-          Title Text: Fatalities/Year by State, Word Wrap = on, font color = FFFFFF, Background color = 262A76, Text Size = 14, Font family = Segoe (Bold)

-          Background: Color = FFFFFF

-          Border: Color = CCCCCC

-          Shadow off

Learn more on how to use the shape map in Power BI
